Projects

Project 4.0

During Project 4.0 we made a solution for smart bins for the city of Herentals, Belgium. It was a group project with students from all the different fields of Computer Science and ICT-Electronics in our school.
Artificial Intelligence Project

During Artificial Intelligence Project we build a game based on Google's Quick,Draw! which is an application that guesses what you are drawing. For this project we created a Convolutional Neural Network and trained it with the data Google offered.
Project Augmented Reality

During this project I created an app with the flutter framework. I also added augmented reality capabilities through wikitude. For example in my application you could display the 3D model of a character of a card game on it's card.
Advanced Programming Topics

During Advanced Programming Topics we learned how to create microservices and how to work with Docker. We also learned how to use some handy tools like SolarCloud to check if your code is fully tested and if there are some potential bugs.

Design Project

During Design Project we learned how to create a data model for a website. The goal of this project was to make a requirement analysis of a website we created later on for another project.
