Artificial Intelligence Project

AI Project was a group project created with 2 other students. During Artificial Intelligence Project we made an application based on the Quick, Draw! website from Google. It's a pictionary game where you get an object to draw and the computer guesses what you are drawing. For this project we created a CNN and trained it in Google Colab with the data provided by Google. When we had our CNN created and trained we used the tkinter library to create a user interface in Python. In this project we learned to integrate AI into a project and to work together on a convolutional neural network and training it in Google Colab. It was also the first time we created a user interface in Python.

What did I learn

We created this whole project together in group most of the time it was one programming and the other 2 helping. This means we learned to work as a team. Next to that we also learned a lot of new technologies like how to create a UI for python and how to integrate AI into an application.
