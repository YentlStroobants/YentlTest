Advanced Programming Topics

Advanced Programming Topics was a project in my last year of application development. For this project we worked in groups of 2 students to create 2 microservices with a brank-edge service on top. The project was fully created in Java. After we had our microservices the goal was to create a fully tested application which is deployed to Dockerhub. We also made it run on kubernetes. To monitor our code coverage we used SonarCloud. If you click the github button below you can see the code on github.

What did I learn

During this project I learned how to create microservices in the first place but I also like to say I learned a lot about Github and Github actions. Before this project we only used Github to store our project and do version control. We never used it to automatically deploy something and run tests. We had a very basic knowledge of Github and for me this opened my eyes to see what the possibilities are with Github
