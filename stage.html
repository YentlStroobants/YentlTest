Internship

Increasing information availability based on production data

28/02/2022 - 27/05/2022

Duracell is the world's leading manufacturer of high performance alkaline batteries, specialty cells and rechargeables. Since its foundation in the early 1940s, the company has become an iconic personal power brand, trusted for compact and longer-lasting batteries. On average 63 Duracell products are sold every second, and Duracell is the number one brand recommended by the world's largest manufacturers.

During my internship I completed several projects. The main project was to create a Power BI report to monitor the Overall Equipment Efficiency. I also made a report to monitor the state of a chain they use which is very expensive. But my internship wasn't only Power BI. I created a project in SharePoint where I had to change an existing PowerApp for the stock managment department to the needs of the people who are using it. This PowerApp was linked to a PowerAutomate flow which also needed some change. It needed some extra values and an option to add more than one attachment. I also worked on a project with Maarten (another intern) to get data out of an iHistorian database and several other data sources. This was possible due to a python script however it was a lot of data so it is really slow. We also extracted data from some other sources for the project because they needed data to send to research groups of KULeuven for a project with the government.

In the action plan you will find a full description of each project and the minimum requirements. In the technical documentation you will find a guide for how to complete each project. In the reflection document you will find what I learned from my internship.
